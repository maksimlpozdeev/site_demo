Категориальная палитра OKLCH — последовательность Ван дер Корпута

L и C фиксированы, оттенок H каждого следующего цвета делит пополам самый большой ещё не занятый промежуток на цветовом круге: 0°, 180°, 90°, 270°, 45°, 135°, 225°, 315° … Это двоично-инверсная (van der Corput) последовательность, отмасштабированная на 360°.

Браузер не поддерживает oklch() в CSS — цвета ниже могут отображаться некорректно.

Порядок размещения на круге

Палитра

Клик по образцу копирует его oklch(...) в буфер обмена. Алгоритм: hue(n) = (H₀ + 360° · vdc(n)) mod 360°, где vdc(n) — значение n в последовательности Ван дер Корпута по основанию 2 (биты числа n, записанные в обратном порядке после двоичной точки).
